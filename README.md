## 📖 使い方

### ✅ 前提条件
- クリエイティブモード必須
- デフォルトアイテムはコンパスです

### 🎯 基本的な使い方
1. クリエイティブモードでコンパス（orツールに設定したアイテム）を手に持つ
2. ブロックをタップするとアイテムとして取得できます

### ⚙️ コマンド一覧

#### `/settool [item]`, `/tapcopy:settool [item]`
ツールとして使うアイテムを設定します
- **権限レベル**: オペレーターのみ実行可能
- **使い方**:
  - `/settool` - 手に持っているアイテムをツールに設定
  - `/settool diamond_pickaxe` - 指定したアイテムをツールに設定

#### `/gettool`, `/tapcopy:gettool`
現在のツールアイテムを取得します
- **権限レベル**: 誰でも実行可能
- **制限**: クリエイティブモード内でのみ使用可能
